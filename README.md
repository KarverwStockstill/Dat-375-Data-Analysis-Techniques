# Dat-375-Data-Analysis-Techniques
Data Analysis Techniques 

I applied common data analysis methods using tools such as Excel, SQL, and MySQL Workbench to process, clean, and model datasets. I used MySQL Workbench to evaluate user activity, engagement metrics, and historical data patterns. I also analyzed how datasets are tracked, modified, and verified to help prevent legal or business errors. In addition, I applied random sampling techniques and pivot tables to assess representative datasets, such as historical crime or injury patterns.
